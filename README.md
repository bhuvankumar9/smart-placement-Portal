# Smart Placement Portal

Live Demo: https://resonant-belekoy-018f47.netlify.app/

## Overview

Smart Placement Portal is a full-stack MERN application designed to streamline campus placements, internships, and career development activities. The platform provides role-based access for students and administrators, enabling job and internship management, course enrollment, placement tracking, secure authentication, email notifications, and career opportunities through a modern and responsive interface.

## Features

* Student & Admin Authentication
* Job Management System
* Internship Management
* Course Enrollment
* Placement Tracking
* Redis OTP Verification
* Email Notifications
* JWT Authentication
* MongoDB Atlas Integration
* Responsive Dashboard UI

## Tech Stack

### Frontend

* React.js
* Redux Toolkit
* Tailwind CSS
* Axios

### Backend

* Node.js
* Express.js
* MongoDB Atlas
* Mongoose
* JWT
* Redis
* Nodemailer
